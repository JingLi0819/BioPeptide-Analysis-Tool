# 高通量生物活性肽功能预测与可视化分析工具

![项目主界面](./be17e58a1b7ea1c984c123fe716b533e.png)

> 基于BIOPEP-UWM数据库的本地化肽序列分析工具，支持批量处理、交互式筛选和可视化展示

## 🚀 项目概述

本项目为小龙虾壳肽研究团队开发的自动化生物活性肽功能分析工具，能够批量处理约700条肽序列，自动识别生物活性功能，并提供交互式的数据筛选和可视化功能。通过本地化部署，确保数据安全性和分析效率。

## ✨ 核心功能

- 🔬 **批量数据导入**：支持Excel/CSV文件和文本输入，一键上传数百条肽序列
- 🧬 **自动化功能分析**：基于BIOPEP-UWM数据库，精准识别96种生物活性功能
- 📊 **交互式数据筛选**：多维度筛选，快速定位目标活性片段
- 📈 **数据可视化**：柱状图、饼图等多种图表直观展示分析结果
- 💾 **结果导出**：CSV格式导出详细分析报告，便于后续研究

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Node.js 16.0+
- 现代Web浏览器（推荐Chrome、Firefox）

### 安装步骤

1. **克隆项目**
```bash
git clone [your-repo-url]
cd peptide-analyzer
```

2. **安装Python依赖**
```bash
pip install -r requirements.txt
```

3. **安装前端依赖**
```bash
npm install
```

4. **启动服务**

```bash
# 启动后端API服务器（终端1）
python peptide_api.py

# 启动前端开发服务器（终端2）
npm run dev
```

5. **访问应用**
打开浏览器访问：http://localhost:5173

## 📋 使用指南

### 1. 数据输入
- 在文本框中直接输入肽序列，每行一个序列
- 或点击"上传Excel/CSV文件"批量导入数据
- 支持标准FASTA格式和简单文本格式

### 2. 开始分析
- 点击"开始分析"按钮，系统将自动处理
- 实时显示分析进度和结果统计

### 3. 结果筛选
- 在"活性功能筛选"区域选择感兴趣的生物活性
- 支持多选组合筛选
- 实时更新可视化图表和结果表格

### 4. 数据导出
- 点击"导出结果"下载CSV格式报告
- 包含原始肽序列、活性片段和生物活性功能

## 🔧 配置说明

### BIOPEP数据库配置
- 默认使用本地`biopep.fasta`文件
- 支持更新到最新BIOPEP-UWM数据库
- 数据文件格式：FASTA格式，包含ID、活性和序列信息

### API端口配置
- 后端服务默认端口：5000
- 前端开发服务器端口：5173
- 支持通过配置文件修改端口设置
