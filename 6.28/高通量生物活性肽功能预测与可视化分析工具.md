# 高通量生物活性肽功能预测与可视化分析工具

## 项目概述

本项目为小龙虾壳肽研究团队开发的自动化生物活性肽功能分析工具，能够批量处理约700条肽序列，自动识别生物活性功能，并提供交互式的数据筛选和可视化功能。

## 主要功能

- ✅ **批量数据导入**：支持Excel/CSV文件和文本输入
- ✅ **自动化功能分析**：基于BIOPEP-UWM数据库的本地化分析
- ✅ **交互式数据筛选**：支持96种生物活性功能筛选
- ✅ **数据可视化**：多种图表展示分析结果
- ✅ **结果导出**：CSV格式导出分析报告

## 技术栈

- **前端**：React 18 + Vite + Tailwind CSS
- **后端**：Flask + Python 3.8+
- **数据处理**：Pandas + 自定义算法
- **可视化**：纯CSS实现的响应式图表

## 快速开始

### 1. 环境要求

- Python 3.8+
- Node.js 16.0+
- 现代Web浏览器

### 2. 安装依赖

```bash
# 安装Python依赖
pip install flask flask-cors pandas requests beautifulsoup4

# 安装前端依赖
cd peptide-analyzer
npm install
```

### 3. 启动服务

```bash
# 启动后端API服务器
python peptide_api.py

# 启动前端开发服务器（新终端）
cd peptide-analyzer
npm run dev
```

### 4. 访问应用

打开浏览器访问：http://localhost:5173

## 使用指南

1. **输入肽序列**：在文本框中输入肽序列或上传Excel/CSV文件
2. **开始分析**：点击"开始分析"按钮
3. **查看结果**：浏览可视化图表和详细分析结果
4. **筛选数据**：选择感兴趣的生物活性功能进行筛选
5. **导出结果**：点击"导出结果"下载CSV格式报告

## 常见问题

**Q: 分析结果为空怎么办？**
A: 检查输入序列格式，确保只包含标准氨基酸字母（A-Z）。

**Q: 系统响应缓慢怎么办？**
A: 减少单次处理的序列数量，建议每次不超过100个序列。

**Q: 如何解读分析结果？**
A: 每行结果包含原始肽序列、活性片段和对应的生物活性功能。
